## CineTrailer

## Descrição do Projeto
O CineTrailer é um aplicativo voltado para amantes de filmes e séries que desejam descobrir novas produções e acompanhar lançamentos de forma prática e interativa. 

O aplicativo deve reunir trailers, sinopses, avaliações e informações sobre cada obra, permitindo ao usuário explorar diferentes gêneros e encontrar rapidamente algo que combine com seu gosto.

Será comsumida a API externa The Movie DB (https://www.themoviedb.org/)
